3.1. Введение в Frontend Разработку

Frontend разработка отвечает за создание интерфейса пользовательской части веб-сайта или веб-приложения. Она включает в себя работу с HTML, CSS и JavaScript.

3.2. Языки Frontend Разработки

Основными языками Frontend разработки являются:

  • HTML (HyperText Markup Language) - используется для структурирования содержимого веб-страницы;
  • CSS (Cascading Style Sheets) - используется для стилизации веб-страницы и создания ее внешнего вида;
  • JavaScript - используется для добавления интерактивности на веб-страницу, такой как анимации, обработка событий и динамическое обновление содержимого.

3.3. Инструменты Frontend Разработки

Некоторые из популярных инструментов для Frontend разработки:

  1. Visual Studio Code - Мощный редактор кода с поддержкой множества языков и плагинов.
  2. React - библиотека для создания пользовательских интерфейсов.
  3. Sass - препроцессор CSS с расширенными возможностями.
  4. Webpack - Сборщик модулей для JavaScript приложений.

3.4. Статьи

Многие компании ищут не просто frontend-программистов, навыки которых ограничиваются написанием кодов SPA (одностраничных приложений, для которых не нужно обновлять страницы), а тех, у кого более широкий кругозор. Чтобы фронтенд разработчик оставался востребованным, он должен теперь уметь следующее: Производить настройку окружения. Сюда включена самостоятельная подготовка компьютера к работе. А еще – установка всех необходимых программ и расширений для локального запуска рабочего приложения. Создание деплоев проектов. Это размещение исполняемого кода на сервере, в пределах которого будет работать проект. Это заключительный этап программирования. Настройка CI/CD. Речь идет о процессе безостановочной сборки и доставки исходного кода до тех или иных сред.
